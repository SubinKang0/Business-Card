# Business Card Generator - Design Spec

## Overview

폼을 입력하면 명함을 생성해주는 웹 애플리케이션. 조코딩AX파트너스 / 대모산개발단 두 가지 템플릿을 지원하며, 이미지(PNG) 및 PDF 다운로드 기능을 제공한다.

## Tech Stack

- **Framework**: Next.js 16 (App Router, TypeScript)
- **Styling**: Tailwind CSS 4
- **Font**: Pretendard (Bold, Semibold)
- **이미지 변환**: html-to-image (DOM → PNG)
- **PDF 생성**: jsPDF (92x52mm CMYK 근사 변환)
- **Package Manager**: pnpm

## Page Structure

단일 페이지(`/`)에 모든 요소 배치.

```
┌─────────────────────────────────────────┐
│  [조코딩AX파트너스]  [대모산개발단]  탭   │
├──────────────────┬──────────────────────┤
│                  │                      │
│   입력 폼        │   명함 미리보기       │
│                  │   (실시간 반영)       │
│  - 이름          │                      │
│  - 직책          │                      │
│  - 전화번호      │                      │
│  - 이메일        │                      │
│                  │                      │
├──────────────────┴──────────────────────┤
│  [이미지 다운로드 (PNG)]  [PDF 다운로드]  │
└─────────────────────────────────────────┘
```

- 탭 전환 시 해당 템플릿 레이아웃+고정 요소로 변경, 입력값은 유지
- 폼 입력값 변경 시 미리보기 실시간 반영

## Templates

### 1. 조코딩AX파트너스

| 위치 | 요소 | 타입 |
|------|------|------|
| 좌상단 | 로고 (`조코딩_로고.svg`) | 고정 |
| 우측 | 심볼 그래픽 (`조코딩_아이콘_심볼.svg`) | 고정 |
| 좌측 중앙 | 이름 | **폼 입력** |
| 좌측 중앙 하 | 직책 | **폼 입력** |
| 좌측 하단 | 주소 아이콘(`조코딩_아이콘_맵.svg`) + "서울 서초구 사평대로 55길 65-7 1층" | 고정 |
| 좌측 하단 | 메일 아이콘(`조코딩_아이콘_메일.svg`) + 이메일 | **폼 입력** |
| 좌측 하단 | 전화 아이콘(`조코딩_아이콘_전화.svg`) + 전화번호 | **폼 입력** |

- 이메일 도메인: `@jocodingax.ai`
- 브랜드 색상: 보라 `#6d319d`, 노랑 `#fbb03b`

### 2. 대모산개발단

| 위치 | 요소 | 타입 |
|------|------|------|
| 좌상단 | 로고 (`대모산개발단_로고.svg`) | 고정 |
| 우상단 | QR코드 (`대모산개발단_QR.svg`) | 고정 |
| 좌측 중앙 | 이름 | **폼 입력** |
| 좌측 중앙 하 | 직책 | **폼 입력** |
| 좌측 하단 | 전화 아이콘(`대모산개발단_아이콘_전화.svg`) + 전화번호 | **폼 입력** |
| 좌측 하단 | 메일 아이콘(`대모산개발단_아이콘_메일.svg`) + 이메일 | **폼 입력** |
| 우측 하단 | 유튜브 아이콘(`대모산개발단_아이콘_유튜브.svg`) + "대모산개발단" | 고정 |
| 우측 하단 | 주소 아이콘(`대모산개발단_아이콘_지도.svg`) + "서울 서초구 사평대로 55길 65-7 1층" | 고정 |

- 이메일 도메인: `@demodev.io`
- 브랜드 색상: 파랑 `#3026b3`

## Typography

- **폰트**: Pretendard
- **자간**: 폰트 사이즈의 2.5%
- **간격(일러 기준)**: 25pt (92x52mm 기준)

| 요소 | 일러 기준 사이즈 | Weight |
|------|-----------------|--------|
| 이름 | 18pt | Bold (700) |
| 직책 | 10pt | Semibold (600) |
| 전화번호 | 6.5pt | Semibold (600) |
| 이메일 | 6.5pt | Semibold (600) |

**일러 기준**: 배경 사이즈 92x52mm 기준의 pt 값. 웹 미리보기에서는 비율을 유지하되 보기 적당한 크기로 환산.

## Download

### 이미지 다운로드 (PNG)

- 해상도: **1088px x 615px**
- 색상: RGB (웹용 고해상도)
- 라이브러리: `html-to-image`
- DOM 요소를 캡처하여 PNG로 변환

### PDF 다운로드

- 크기: **92mm x 52mm**
- 색상: CMYK 변환 (RGB → CMYK 근사 변환)
- 라이브러리: `jsPDF`
- 인쇄용 고해상도 출력

## Form Fields

| 필드 | 타입 | 비고 |
|------|------|------|
| 이름 | text | 한글 2~5자 |
| 직책 | text | 예: COO / 부대표, CCO |
| 전화번호 | tel | 형식: 010 0000 0000 |
| 이메일 | email | 도메인은 템플릿에 따라 고정 |
